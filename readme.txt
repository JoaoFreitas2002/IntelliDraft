== Installation ==

1. Upload the `IntelliDraft` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the IntelliDraft settings page to configure the plugin and connect it to your AI API key.
